# Boundaries Definition

## Update Boundary

### Función
Controlar cambios internos del sistema

### Hace
- Aplicar propuestas
- Validar reglas
- Registrar cambios

### No hace
- No interpreta inputs externos
- No ejecuta side effects

---

## Execution Boundary

### Función
Controlar acciones externas

### Hace
- Ejecutar acciones externas
- Validar permisos finales
- Registrar ejecución

### No hace
- No decide qué hacer
- No interpreta negocio
